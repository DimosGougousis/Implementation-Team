## Purpose

Implementation Team OS — tracks Temenos Transact implementation programmes through an 8-stage lifecycle with contribution logging and rich artefact templates.

## Structure

- `governance/` — Framework and directives
- `agents/` — Agent definitions (placeholders)
- `artefacts/` — 8 artefact template folders
- `lifecycle/` — 8-stage lifecycle with gates
- `pipeline/` — Per-programme tracking
- `analytics/` — KPIs and reporting
- `team/` — Team directory

## Commands

- `/new-programme <name>` — Scaffold a new programme pipeline
- `/impl-status` — Cross-programme status dashboard
