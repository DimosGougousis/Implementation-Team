# Pre-Cutover Readiness Checklist

## Data Migration

- [ ] All data domains profiled and cleansed
- [ ] Mapping specifications complete and reviewed
- [ ] Rehearsal 1 completed with acceptable results
- [ ] Rehearsal 2 completed with acceptable results
- [ ] Quality gates passed for all domains
- [ ] Reconciliation scripts tested

## Testing

- [ ] UAT sign-off obtained
- [ ] All Critical defects resolved
- [ ] All High defects resolved or mitigated
- [ ] Performance benchmarks met
- [ ] Security testing completed

## Infrastructure

- [ ] Production environment provisioned
- [ ] Network connectivity verified
- [ ] SSL certificates installed
- [ ] Monitoring and alerting configured
- [ ] Backup and recovery tested

## Integration

- [ ] All integrations deployed to production
- [ ] Integration smoke tests passed
- [ ] API gateway configured
- [ ] Event channels active

## Cutover

- [ ] Cutover runbook reviewed and approved
- [ ] Cutover rehearsal completed
- [ ] Rollback procedure tested
- [ ] Go/no-go criteria documented
- [ ] Communication plan approved

## People

- [ ] War room booked
- [ ] All participants confirmed
- [ ] Vendor support on standby
- [ ] Escalation contacts identified
- [ ] Support team trained
